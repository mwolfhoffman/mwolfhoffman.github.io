Stuff I have made.

Client Projects

Shuffle

Summary: Create a team, then create randomized groups. Perfect for classrooms.
Tech Used: Vue, Vuex, Node, MongoDB, Webpack
In this full-stack project I created my own local authentication using email and password to gain a better understanding of authentication. This was also my first mongoDB project. I used bcrypt JS for secure password hashing.

Boise CodeWorks Backpack

Summary: Online portal for coding bootcamp courses.
Tech Used: Vue, Vuex, Firebase, Pug, Sass.
I worked on a team with 3 other developers to create an online portal for students to complete assignments and register for courses.

Artsy Angler

Summary: Portfolio for a local artist with an online store.
Tech Used: Vue, Node, MongoDB, Keystone, StripeJS, Heroku
Utilizing KeystoneJS for content management, I allowed my client to easily upload artwork, process payments, and share his craft.

Side Projects

The Tip

Summary: This app allows you to enter the amount of a bill and select either a pre-determined tip percentage or create your own custom tip.
Tech Used: React-Native, Android Studio, ES6, Babel
This is my first Android app. It was a project to learn mobile app development and React-Native. It takes advantage of ES6 classes and other novel JavaScript features.

Drone Tracker

Summary: Front-end project using DroneSteam API. This web app allows users to browse US Drone Strikes in Parkistan, Somalia, and Yemen.
Tech Used: HTML/CSS, Materialize CSS, AngularJS, DroneSteam API, Gulp
I used Materialize CSS instead of Bootstrap for UI friendly components and also learned how to work with the Gulp task runner in this project.

Drink Up

Summary: Users browse beers and can organize lists of their favorites, least favorites, or those they wish to try.
Tech Used: AngularJS, BreweryDB API, Gulp, Flexbox
This was my first independent project and my goal was to gain a better understanding of AngularJS.
